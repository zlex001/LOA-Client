# 本地化系统（Localization）

负责客户端的多语言支持，包括语言检测、文本加载、语言切换等功能。

## 系统概述

本地化系统提供：
- **语言检测**：自动检测系统语言和服务器语言
- **文本加载**：加载多语言JSON文件
- **文本获取**：通过Key获取本地化文本
- **动态切换**：运行时切换语言

**核心文件**：
- `Assets/Framework/Localization.cs`：本地化管理器
- `Assets/Framework/LanguageDetector.cs`：语言检测器
- `Assets/Resources/Localization/`：多语言JSON文件

## Localization（本地化管理器）

**Localization** 是本地化的单例管理器，负责加载和管理多语言文本。

### Localization 类定义

```csharp
public class Localization : Singleton<Localization>
{
    private Dictionary<string, string> texts = new Dictionary<string, string>();
    private string currentLanguage;
    
    // 初始化
    public void Init(string language);
    
    // 获取文本
    public string Get(string key, params object[] args);
    
    // 切换语言
    public void ChangeLanguage(string language);
    
    // 标准语言代码
    public string StandardLanguageCode { get; }
}
```

### 初始化（Init）

```csharp
public void Init(string language)
{
    currentLanguage = language;
    LoadLanguageFile(language);
    
    Debug.Log($"[Localization] Initialized: {language}");
}

private void LoadLanguageFile(string language)
{
    // 加载JSON文件
    string path = $"Localization/{language}";
    TextAsset asset = Resources.Load<TextAsset>(path);
    
    if (asset == null)
    {
        Debug.LogError($"[Localization] Language file not found: {path}");
        return;
    }
    
    // 解析JSON
    texts = JsonUtility.FromJson<Dictionary<string, string>>(asset.text);
    
    Debug.Log($"[Localization] Loaded {texts.Count} texts");
}
```

### 获取文本（Get）

```csharp
public string Get(string key, params object[] args)
{
    if (!texts.TryGetValue(key, out string text))
    {
        Debug.LogWarning($"[Localization] Key not found: {key}");
        return $"[{key}]";
    }
    
    // 格式化参数
    if (args.Length > 0)
    {
        try
        {
            text = string.Format(text, args);
        }
        catch (Exception ex)
        {
            Debug.LogError($"[Localization] Format error: {key}, {ex.Message}");
        }
    }
    
    return text;
}
```

**使用示例**：

```csharp
// 简单文本
string welcome = Localization.Instance.Get("welcome");  // "欢迎来到游戏"

// 带参数的文本
string greeting = Localization.Instance.Get("greeting", "玩家名");  
// "你好，玩家名！"

// 多参数
string progress = Localization.Instance.Get("download_progress", 50, 100);
// "已下载 50/100"
```

### 切换语言（ChangeLanguage）

```csharp
public void ChangeLanguage(string language)
{
    if (currentLanguage == language)
    {
        Debug.Log($"[Localization] Already using language: {language}");
        return;
    }
    
    // 加载新语言文件
    LoadLanguageFile(language);
    currentLanguage = language;
    
    // 保存到本地
    PlayerPrefs.SetString("LANGUAGE", language);
    PlayerPrefs.Save();
    
    // 更新Data
    Data.Instance.Language = Enum.Parse<Data.Languages>(language);
    
    Debug.Log($"[Localization] Changed language to: {language}");
}
```

## LanguageDetector（语言检测器）

**LanguageDetector** 负责检测系统语言和服务器语言配置。

### 系统语言检测（DetermineLanguage）

```csharp
public static string DetermineLanguage()
{
    // 获取系统语言
    SystemLanguage systemLanguage = Application.systemLanguage;
    
    // 映射到支持的语言
    switch (systemLanguage)
    {
        case SystemLanguage.Chinese:
        case SystemLanguage.ChineseSimplified:
            return "ChineseSimplified";
        
        case SystemLanguage.ChineseTraditional:
            return "ChineseTraditional";
        
        case SystemLanguage.English:
            return "English";
        
        case SystemLanguage.Japanese:
            return "Japanese";
        
        case SystemLanguage.Korean:
            return "Korean";
        
        case SystemLanguage.French:
            return "French";
        
        case SystemLanguage.German:
            return "German";
        
        case SystemLanguage.Spanish:
            return "Spanish";
        
        case SystemLanguage.Portuguese:
            return "Portuguese";
        
        case SystemLanguage.Russian:
            return "Russian";
        
        default:
            return "English";  // 默认英语
    }
}
```

### 服务器语言检测（DetectFromServer）

```csharp
public static void DetectFromServer(string gateway, Action<string> callback)
{
    string url = $"http://{gateway}:8880/api/language/detect";
    
    Http.Instance.RequestGet(url, (success, response) =>
    {
        if (success)
        {
            try
            {
                var data = JsonUtility.FromJson<LanguageResponse>(response);
                callback(data.language);
            }
            catch (Exception ex)
            {
                Debug.LogError($"[LanguageDetector] Parse failed: {ex.Message}");
                callback(DetermineLanguage());  // 使用系统语言
            }
        }
        else
        {
            Debug.LogWarning("[LanguageDetector] Server request failed");
            callback(DetermineLanguage());  // 使用系统语言
        }
    });
}

[Serializable]
private class LanguageResponse
{
    public string language;
}
```

## 最佳实践

### 1. 使用Key而非硬编码文本

```csharp
// ✅ 推荐：使用Key
text.text = Localization.Instance.Get("welcome");

// ❌ 不推荐：硬编码
text.text = "欢迎来到游戏";
```

### 2. 参数化动态内容

```csharp
// ✅ 推荐：使用参数
Localization.Instance.Get("download_progress", 50, 100);

// ❌ 不推荐：拼接字符串
$"已下载 {downloaded}/{total}";
```

### 3. 缺失Key的处理

```csharp
public string Get(string key, params object[] args)
{
    if (!texts.TryGetValue(key, out string text))
    {
        Debug.LogWarning($"[Localization] Key not found: {key}");
        return $"[{key}]";  // 返回[Key]便于调试
    }
    return text;
}
```

### 4. 切换语言后刷新UI

```csharp
public void ChangeLanguage(string language)
{
    LoadLanguageFile(language);
    currentLanguage = language;
    
    // 触发UI刷新事件
    Game.Event.Instance.Fire(UI.Event.LanguageChanged);
}

// UI监听语言变化
Game.Event.Instance.Add(UI.Event.LanguageChanged, (args) => {
    RefreshTexts();
});
```

## 总结

本地化系统通过以下模块实现多语言支持：

1. **Localization**：本地化管理器，加载和管理多语言文本
2. **LanguageDetector**：语言检测器，自动检测系统和服务器语言

**核心特性**：
- 自动检测：自动检测系统语言
- 动态切换：运行时切换语言
- 参数化文本：支持格式化参数
- 标准化：使用标准语言代码

这种设计确保了游戏的国际化能力和用户体验。
